# Personal Vault

Try it online: [Personal Vault](https://vidul-nikolaev-petrov.github.io/personal-vault/)

## Encryption Steps
1. Enter the text to encrypt.
2. Enter a password.
3. Click the **Encrypt** button.
4. Save the file (`Ctrl+S` or `Cmd+S`).

## Decryption Steps
1. Open the previously saved file.
2. Enter the password you used before.
3. Click the **Decrypt** button.
4. That's it.
